# 范辉昊 · Huihao Fan — Personal Website v2

Nostalgic, deeply interactive academic portfolio with Chinese accents and Hohhot memories.

## Features
- Custom animated cursor
- Loader with Chinese character animation
- Hero with particle canvas (animated gold network)
- Floating lanterns (🏮) rising continuously
- **Interactive Plaza Scene** — day/dusk/night toggle, hoverable memory hotspots
- Scroll-animated timeline
- Filterable publications grid
- Red scroll progress bar
- Parallax hero
- Fully responsive

## Files
- `index.html`
- `style.css`
- `script.js`

## Deploy to GitHub Pages
1. Create a new public repo
2. Upload all 3 files
3. Settings → Pages → main branch / root → Save
4. Live at: `https://YOUR_USERNAME.github.io/REPO_NAME/`
